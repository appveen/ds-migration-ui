Overview
  • In both ODP and Data Stack the Encryption and Decryption is done by App's Key and Cerificate that gets generated on an App's Creation
  • The Key & Cert Provided by customer or the ENCRYPTION_KEY is used to Encrypt & Decrypt the App's Key & Cert.
  • ODP used Key & Cert Provided by customer and mounted as a Secret in K8S.
  • Data Stack uses ENCRYPTION_KEY as text value stored in ENV.
