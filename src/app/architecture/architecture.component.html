Overview
  • ODP and Data Stack both works on top of Kubernetes(K8s)
  • ODP needed a docker/podman as realtime container builder mounted in DM to build images. Where as Data Stack doesn't need it.
  • In Case of B2B ODP had a 2-way SSL communication, which was difficult to configure. In Data Stack we have removed it and made the communication simpler.
